# Чек-лист тестирования веб-приложения Rewatch

## 1. Регистрация пользователя
- Проверка открытия страницы регистрации по адресу `/register`
- Проверка отображения полей Username, Email, Password, Repeat Password
- Проверка невозможности отправки формы с пустым Username
- Проверка невозможности отправки формы с пустым Password
- Проверка невозможности отправки формы с пустым Repeat Password
- Проверка отображения паролей в скрытом виде
- Проверка успешной регистрации при вводе корректных данных
- Проверка появления сообщения об ошибке при несовпадении паролей
- Проверка появления ошибки при регистрации с существующим username
- Проверка появления ошибки при регистрации с существующим email
- Проверка автоматической авторизации после успешной регистрации
- Проверка корректного отображения flash-сообщений без дублирования

---

## 2. Авторизация пользователя

- Проверка открытия страницы авторизации по адресу `/login`
- Проверка входа пользователя по username
- Проверка входа пользователя по email (если предусмотрено)
- Проверка успешной авторизации при вводе корректных данных
- Проверка отображения ошибки при вводе неверного пароля
- Проверка отображения ошибки при вводе несуществующего пользователя
- Проверка невозможности отправки формы с пустыми полями
- Проверка редиректа после успешной авторизации

---

## 3. Выход из системы

- Проверка выполнения выхода по адресу `/logout`
- Проверка завершения пользовательской сессии после выхода
- Проверка недоступности защищённых страниц после выхода
- Проверка корректного редиректа после выхода из системы

---

## 4. Главная страница

- Проверка открытия главной страницы без ошибок
- Проверка отображения списка популярных/трендовых тайтлов
- Проверка отображения карточек с названием и постером
- Проверка отображения заглушки при отсутствии постера
- Проверка корректной работы страницы при недоступности внешнего API

---

## 5. Каталог

- Проверка открытия страницы каталога по адресу `/catalog`
- Проверка отображения списка фильмов и сериалов
- Проверка работы фильтрации по типу (all / movie / tv)
- Проверка визуального выделения активного фильтра
- Проверка корректной работы пагинации
- Проверка обработки невалидных параметров URL
- Проверка перехода на страницу тайтла при клике по карточке

---

## 6. Поиск

- Проверка доступности поля поиска
- Проверка поиска по существующему названию
- Проверка отображения пустого результата при отсутствии совпадений
- Проверка обработки пустого поискового запроса
- Проверка отображения в результатах только фильмов и сериалов
- Проверка перехода на страницу тайтла из результатов поиска

---

## 7. Страница тайтла

- Проверка открытия страницы тайтла по адресу `/title/<media_type>/<id>`
- Проверка ограничения доступа для неавторизованных пользователей
- Проверка отображения основной информации о тайтле
- Проверка отображения постера или заглушки
- Проверка сохранения пользовательской заметки
- Проверка сохранения статуса просмотра
- Проверка невозможности сохранения невалидного статуса
- Проверка корректного отображения flash-сообщений

---

## 8. Прогресс сериалов

- Проверка отображения списка сезонов и эпизодов
- Проверка сохранения прогресса просмотра сезона
- Проверка невозможности сохранения отрицательного значения просмотренных эпизодов
- Проверка невозможности превышения количества эпизодов сезона
- Проверка добавления заметки к эпизоду
- Проверка удаления заметки к эпизоду

---

## 9. Мой список

- Проверка открытия страницы `/my_list`
- Проверка ограничения доступа для неавторизованных пользователей
- Проверка отображения тайтлов, добавленных пользователем
- Проверка работы фильтров списка
- Проверка работы сортировки списка
- Проверка перехода на страницу тайтла из списка

---

## 10. Статистика

- Проверка открытия страницы статистики по адресу `/stats`
- Проверка ограничения доступа к странице статистики
- Проверка корректности подсчёта общего количества тайтлов
- Проверка корректности расчёта среднего рейтинга
- Проверка корректности отображения статистики по статусам
- Проверка отсутствия ошибок при отсутствии пользовательских данных

---

## 11. Профиль пользователя

- Проверка открытия страницы профиля по адресу `/profile`
- Проверка отображения имени пользователя
- Проверка отображения email пользователя (если предусмотрено)
- Проверка отображения аватара пользователя
- Проверка использования аватара по умолчанию при его отсутствии

---

## 12. Редактирование профиля

- Проверка открытия страницы редактирования по адресу `/profile/edit`
- Проверка возможности изменения username
- Проверка невозможности сохранения занятого username
- Проверка возможности изменения email
- Проверка возможности изменения пароля
- Проверка отсутствия изменения пароля при пустом значении
- Проверка загрузки аватара допустимого формата
- Проверка отклонения аватара недопустимого формата
- Проверка отклонения аватара превышающего допустимый размер
- Проверка сохранения изменений профиля
