# Тест-кейсы для веб-приложения Rewatch

## TC-001

**Идентификатор:** TC-001  
**Заголовок:** Успешная регистрация пользователя  

**Предусловия:**  
Открыта страница регистрации `/register`. Пользователь с указанным username и email отсутствует в системе.

**Шаги выполнения и ожидаемый результат:**

1. Ввести корректный Username  
   **Ожидаемый результат:** Поле Username принимает введённое значение без ошибок  

2. Ввести корректный Email  
   **Ожидаемый результат:** Поле Email принимает введённое значение  

3. Ввести пароль  
   **Ожидаемый результат:** Пароль отображается в скрытом виде  

4. Повторить пароль  
   **Ожидаемый результат:** Повторный пароль введён корректно  

5. Нажать кнопку регистрации  
   **Ожидаемый результат:** Пользователь зарегистрирован, автоматически авторизован, выполнен редирект на главную страницу  

---

## TC-002

**Идентификатор:** TC-002  
**Заголовок:** Регистрация с несовпадающими паролями  

**Предусловия:**  
Открыта страница регистрации `/register`.

**Шаги выполнения и ожидаемый результат:**

   **Ожидаемый результат:** Username введён без ошибок  

2. Ввести Email  
   **Ожидаемый результат:** Email введён без ошибок  

3. Ввести пароль  
   **Ожидаемый результат:** Пароль введён  

4. Ввести другой пароль в поле Repeat Password  
   **Ожидаемый результат:** Значения паролей не совпадают

   **Ожидаемый результат:** Регистрация не выполнена, отображается сообщение об ошибке о несовпадении паролей
